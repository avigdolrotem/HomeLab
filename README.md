# HomeLab

**Personal AWS HomeLab: self-hosted services, DevOps automation, and infrastructure-as-code for practical cloud skills.**

This project documents my journey building a personal HomeLab on AWS. The goal is to deploy and manage self-hosted services—like a password manager, backup cloud, and custom mailbox—using modern DevOps practices and infrastructure-as-code (Terraform).

## 🛠️ Tech Stack

- AWS
- Terraform
- Docker
- Linux
- SSL/HTTPS
- More coming soon!

## 🎯 Project Goals

- Learn real-world cloud infrastructure and automation
- Practice secure deployments and networking
- Build a reusable, scalable HomeLab environment
- Share milestones and lessons on LinkedIn

## 🚩 Milestones

- [V] Custom mailbox
- [ ] Password manager (Vaultwarden)
- [ ] Cloud backup for photos/files
- [ ] (Planned) Remote code editor (VS Code Server)
- [ ] (More to come)
